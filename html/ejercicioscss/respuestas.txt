EJERCICIO 1

○ ¿Qué diferencias encontraste entre los tres métodos de aplicar CSS?
Para mí, el CSS en línea es útil cuando quieres aplicar el estilo a algo puntual, es decir, a un párrafo o un título, pero si quisiera tener el mismo estilo para todos los párrafos, elegiría el CSS Interno. Pese a esto, mi preferido es el CSS Externo ya que me parece más completo y sobretodo más limpio, ya que todos los diseños quedan fuera del fichero y puedo trabajar mejor en el código con menos distracciones.

○ ¿Qué ventajas ves en usar CSS externo frente a los otros métodos?
Lo mencionado antes, la limpieza y también la comodidad para tener todos los estilos agrupados en un mismo fichero css.
